Schooledule representa una solucion robusta y profesional para la gestion de centros educativos. La combinacion de Java 21, Spring Boot y un modelo de datos multi-sede ha permitido superar las limitaciones de rigidez de los sistemas actuales.

El proyecto ha cumplido los objetivos principales:
\begin{itemize}
    \item Gestion multi-sede garantizada mediante el aislamiento de datos.
    \item Integridad inmutable de las calificaciones a traves de la auditoria forense.
    \item Un ciclo de vida de desarrollo moderno con CI/CD y purga de herramientas.
\end{itemize}
