Schooledule representa una soluci\'{o}n robusta y profesional para la gesti\'{o}n de centros educativos. La combinaci\'{o}n de Java 21, Spring Boot y un modelo de datos multi-sede ha permitido superar las limitaciones de rigidez de los sistemas actuales.

El proyecto ha cumplido los objetivos principales:
\begin{itemize}
    \item Gesti\'{o}n multi-sede garantizada mediante el aislamiento de datos.
    \item Integridad inmutable de las calificaciones a trav\'{e}s de la auditor\'{i}a forense.
    \item Un ciclo de vida de desarrollo moderno con CI/CD y purga de herramientas.
\end{itemize}
